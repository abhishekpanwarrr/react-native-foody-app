## Run Locally

Clone the project

```bash
  git clone https://github.com/abhishekpanwarrr/react-native-foody-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run ios 
  npm run android
```
